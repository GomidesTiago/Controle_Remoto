# Simulador Controle Remoto
o projeto simula o funcionamento de um controle remoto com as funções
- Liga
- Desliga
- Abrir Menu
- Fechar Menu
- Aumentar Volume
- Diminuir Volume
- Ligar Mudo
- Desligar Mudo
- Play
- Pause;


## Uso pra estudo
O intuito do projeto foi estudar encapsulamento, interface, implements.
